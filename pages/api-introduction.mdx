## Introduction

Welcome to the Envlink SaaS URL Shortener API documentation. Our API provides a robust and efficient solution for creating, managing, and tracking shortened URLs. It is designed to be both powerful and easy to use, catering to a wide range of applications, from simple link shortening to complex analytics and subscription management.

This documentation will guide you through the various features and endpoints available in our API. Whether you are a developer looking to integrate our services into your application or seeking to understand the full capabilities of our platform, you will find all the necessary information here.

## Core Features

Our API is built around a comprehensive set of features that allow you to manage the entire lifecycle of your shortened URLs. Here are some of the key functionalities you can expect:

- **Account Management**: Our API offers a comprehensive suite of account management features, allowing you to handle user authentication and settings with ease. You can register, log in, and manage user profiles, including password changes and email verification. All of these features are designed to be secure and straightforward, ensuring a seamless user experience. [Learn more about the Account API.](/account)

- **URL Shortening**: The core of our service is the ability to shorten long URLs into a more manageable format. You can create custom short codes, set expiration dates, and even protect your links with an access code. This flexibility allows you to tailor your shortened URLs to your specific needs, whether for public or private use. [Learn more about the URL Shortening API.](/urls)

- **Analytics**: Understanding how your links are being used is crucial for any application. Our API provides detailed analytics, including click counts, unique visitors, and geographic data. You can also access timeline-based statistics to monitor trends and performance over time. [Learn more about the Analytics API.](/analytics)

- **Subscriptions and Billing**: For our premium users, we offer a range of subscription plans with varying features and limits. The API allows you to manage your subscriptions, upgrade or downgrade your plan, and access your billing history. This ensures that you can always choose the plan that best fits your needs. [Learn more about the Subscriptions API.](/subscriptions)

- **Payment Methods**: To make our subscription process as smooth as possible, we support a variety of payment methods. You can add, update, and manage your payment details securely through the API, ensuring that your account is always in good standing. [Learn more about the Payment Methods API.](/payment-methods)

- **Webhooks**: Our API also includes support for webhooks, allowing you to receive real-time notifications about important events. This is particularly useful for tracking subscription updates, payment confirmations, and other critical information without having to poll the API continuously. [Learn more about the Webhooks API.](/webhooks)

- **Authentication**: Our API provides a secure and straightforward way to handle user registration, login, and session management. [Learn more about the Authentication API.](/authentication)

- **Billing Address**: The Billing Address API is designed to manage user billing information in a secure and organized manner. [Learn more about the Billing Address API.](/billing-address)

- **Health**: The Health API is a straightforward endpoint that allows you to monitor the health and status of the application. [Learn more about the Health API.](/health)

- **OAuth**: The OAuth API provides a secure and seamless way for users to authenticate with their existing accounts from third-party providers, such as Google. [Learn more about the OAuth API.](/oauth)

- **Public Payment Methods**: The Public Payment Methods API is designed to allow users to add and manage their payment methods without being authenticated. [Learn more about the Public Payment Methods API.](/public-payment-methods)

- **Public Subscriptions Plans**: The Public Subscriptions Plans API allows you to retrieve information about the available subscription plans without requiring authentication. [Learn more about the Public Subscriptions Plans API.](/public-subscriptions-plans)

- **Public URLs**: The Public URLs API provides a simple and efficient way to shorten URLs without requiring user authentication. [Learn more about the Public URLs API.](/public-urls)

- **Sessions**: The Sessions API is a critical component for managing user authentication and security. [Learn more about the Sessions API.](/sessions)

- **Transactions**: The Transactions API allows you to retrieve detailed information about user transactions, providing a complete history of all payments, refunds, and other billing-related events. [Learn more about the Transactions API.](/transactions)

- **User**: The User API provides a straightforward way to manage user profile information. [Learn more about the User API.](/user)

We are committed to providing a reliable and feature-rich service that meets the demands of modern applications. As you explore our documentation, you will discover the full extent of what our API has to offer.
