# Envlink API Documentation

Welcome to the Envlink SaaS URL Shortener API documentation. Our API provides a robust and efficient solution for creating, managing, and tracking shortened URLs. It is designed to be both powerful and easy to use, catering to a wide range of applications, from simple link shortening to complex analytics and subscription management.

This documentation will guide you through the various features and endpoints available in our API.

## API Reference

The following is a list of available API documentation. All documentation is currently a work in progress.

### Core APIs
*   **Account API**: Manage user authentication, profiles, and security. ([Details](pages/account.mdx) - *Soon*)
*   **Analytics API**: Get detailed statistics and tracking for your short URLs. ([Details](pages/analytics.mdx) - *Soon*)
*   **Authentication API**: Handle user registration, login, and token management. ([Details](pages/authentication.mdx) - *Soon*)
*   **URLs API**: Create, retrieve, update, and delete short URLs. ([Details](pages/urls.mdx) - *Soon*)
*   **User API**: Manage user profile information. ([Details](pages/user.mdx) - *Soon*)
